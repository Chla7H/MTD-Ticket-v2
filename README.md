# Mega Team ® Ticket Bot V2

An advanced and secure Discord Ticket Bot built with `discord.js v14`. It features custom Discord emojis and three streamlined ticket departments: **Buy Anti-Cheat**, **Apply Staff**, and **Reports/General Support**.

---

## Getting Started

Follow these step-by-step instructions to get your ticket bot running up and fully functional.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v16.11.0 or higher) installed on your operating system.

### Installation & Configuration

1. **Download/Extract** all files into a single project folder.
2. Open your project directory in your terminal or Command Prompt (CMD).
3. Install the required Node.js package by executing:
